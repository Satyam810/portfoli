## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/Satyam810/satyam-portfolio.git
cd satyam-portfolio

# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
satyam-portfolio/
├── app/                  # Next.js App Router (layout, page, globals.css)
├── components/
│   ├── main/             # Section-level components (Hero, Skills, Projects, etc.)
│   └── sub/              # Reusable sub-components (HeroContent, EventCard, etc.)
├── constants/            # Shared data (skills, nav links, socials, certifications)
├── config/               # Site metadata & SEO config
├── data/                 # Projects and events data
├── lib/                  # Utility functions and motion variants
├── public/               # Static assets (images, videos, certifications, skills)
└── types/                # TypeScript type declarations
```
